# 股票代码

- 600 开头的股票是上证 A 股， 属于大盘股，其中 `6006` 开头的股票是 最早上市的股票，`6016` 开头的股票为大盘蓝筹股；
- 900 开头的股票是上证 B 股；
- 000 开头的股票是深证 A 股，`001`、`002` 开头的股票也都属于深证 A 股，其中 `002` 开头的股票是深证 A 股中小企业股票；
- 200 开头的股票是深证 B 股；
- 300 开头的股票是创业板股票；
- 400 开头的股票是三板市场股票
